# LV2 점프와 순간 이동

### 1차 시도 (성공)
```py
def solution(n):
    ans = 0
    while n>0:
        ans += n%2
        n = n // 2
    return ans
```
> 순간이동에는 건전지 사용량이 줄지 않기 때문에 최대한 많은 순간이동을 해야 한다. 또한 순간이동은 현재 이동 거리의 두 배만큼 가능하기에, 마지막 도착 지점에서 역으로 계산하여 순간이동이 가능한 지점을 계산하고 그 자리에서 2의 배수가 되는 가장 가까운 거리로 점프한다면 그 이동량이 최소 이동량인 것을 알 수 있다.
